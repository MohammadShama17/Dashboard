# Dashboard Project 

A modern and responsive **Dashboard interface** built using **HTML and CSS**.  
Designed to display key information and analytics in a clear and visually appealing layout.

---

## Features
-  Responsive layout for desktop and tablet screens
-  Sidebar navigation
-  Interactive cards/widgets
-  Clean and modern design
-  Fully customizable UI elements

---

## How to Use
1. Open `index.html` in your web browser.
2. Navigate through the dashboard using the sidebar.
3. Customize elements or add your own content in `index.html` and `style.css`.

---

## Technologies Used
- **HTML5** – Structure of the dashboard
- **CSS3** – Styling, layout, and responsive design
